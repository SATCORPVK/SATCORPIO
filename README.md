# SATCORP – Complete Skill Deck Website

A modern, responsive, single‑page (or multi‑page) site that showcases SATCORP’s concierge‑level creative & systems architecture services.

## Features
- Clean, semantic HTML5 + BEM‑style CSS (no external framework needed)
- Dark / Light mode toggle (persists in localStorage)
- Feather icons, smooth scrolling, responsive card grid
- Easy‑to‑replace contact form (Formspree or custom endpoint)
- Ready for deployment on GitHub Pages, Netlify, Vercel
